How to Embed

Add this single line of code to your website, just before the closing </body> tag:

<script src="http://localhost:3000/embed/widget.js"></script>

Customization Options

You can customize the widget using URL parameters:

<script src="http://localhost:3000/embed/widget.js?position=bottom-left&color=%23667eea&header=Help%20Center"></script>

Available Parameters:

  1. position - "bottom-right" or "bottom-left"
  2. color - Primary color (URL encoded hex, e.g., %23667eea)
  3. header - Header text (URL encoded)
  4. welcome - Welcome message (URL encoded)
  5. requireName - "true" to require customer name
  6. requireEmail - "true" to require customer email

JavaScript API

Control the widget programmatically:

// Open the widget
ChatWidget.open();

// Close the widget
ChatWidget.close();

// Toggle the widget
ChatWidget.toggle();
